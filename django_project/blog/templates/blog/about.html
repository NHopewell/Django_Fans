About Django Fans

Hi there, Nick here. Thanks for checking out Django Fans.

This is a CRUD project that I wanted to complete to help improve my general development skills. I wanted to make some kind of forum as creating online forums require implementing core concepts of web development. I ultimately chose to make the forum about the backend framework I used for the project, Django.

In order to really improve my skills, I wanted to make this forum a lot cleaner than most forums I have seen listed on the project portfolios of other developers. I wanted it to include some nice features one might expect from a proper forum they frequent. Although I could certaintly add a lot more, I hope the time I put in shows.
